Lab: Delegates and Events .NET

Create Console programs for the following exercises:

Exercise 1

Create a class LightBulb, with a bool property IsOn and a string property Color. Now create a class called Switch where you implement methods TurnOn and TurnOff. Use only delegates to let a LightBulb instance subscribe to a Switch instance, so that the LightBulb turns on using the Switch.

Exercise 2

Based on the previous exercise, use multiple switches, so that one of the switches should turn on in order to put the light on.

Exercise 3

Now use events to implement the previous exercises.

Exercise 4

Create a TemperatureSensor class that measures temperature and raises an event every second, with a new temperature reading. Create 4 subscriber classes:

  • AverageTemperature : average of all readings
  • MinimumTemperature : minimum of all readings
  • MaximumTemperature : maximum of all readings
  • TresholdTemperature : raises an event itself when a treshold temperature is reached. E.g. if it gets hotter dan 45 degrees, an alarm gets off

Exercise 5

Proof using any of the examples in the previous exercises, that a delegate return value is actually the last method the invocation list.
